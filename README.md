# Spicetify Auto Installer (Windows)

A simple, noob-friendly batch script that automatically installs and sets up
Spicetify for Spotify on Windows.

This project is intended to make the Spicetify installation process easier for
beginners by automating the most common steps.

---

## Features

- Automatic Spicetify installation
- No command-line knowledge required
- Beginner-friendly `.bat` script
- Works with the official Spotify desktop app (Windows)

---

## Requirements

- Windows 10 or newer
- Spotify desktop app installed
- Internet connection

---

## How to Use

1. Download the latest release from this repository
2. Right-click the `.bat` file and select **Run as administrator**
3. Follow the on-screen instructions
4. Restart Spotify once the script finishes
